如何找到小账本

1、公众号“微信收款助手”

关注公众号“微信收款助手” → 点击公众号菜单栏“小账本”

2、微信收付款入口

微信右上角+ → 点击收付款 → 进入二维码收款 → 点进下方“收款小账本”

3、微信搜索“收款小账本”

进入小账本后，推荐点击页面右上角的“...”-“添加到我的小程序”
